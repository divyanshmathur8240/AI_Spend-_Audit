# METRICS

## North Star
Qualified audit completions that produce a lead and savings insight.

## Input metrics
- Number of completed audits
- Email capture conversion rate
- Shareable report visits

## Instrumentation
- Track audit completion events, tool selections, and savings thresholds.
- Measure email capture and consultation-booking clicks.

## Pivot trigger
If audit-to-lead conversion falls below 2%, reconsider the value proposition or onboarding flow.
